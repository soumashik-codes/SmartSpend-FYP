# SmartSpend – Final Year Project

SmartSpend is a full-stack personal finance web application developed as a Final Year Project.

## Architecture
- **Backend:** FastAPI (Python), SQLAlchemy, SQLite
- **Frontend:** Next.js (React, TypeScript, Tailwind CSS)
- **Architecture Style:** Decoupled frontend–backend (REST API)

## Core Features (Current)
- CSV bank statement upload
- Robust transaction parsing (credit/debit, money in/out)
- Secure backend transaction storage
- Dashboard with income, expenses, and balance summary

## Status
Post-IPD development in progress.
